在这里摘录一下官方文档中的配置参数：

#Site

设置项 描述
title 站点的标题
subtitle 站点的副标题
description 站点描述，会被一些主题用到
author 你的名字。
language 网站使用的语言，[使用2个字符表示][8]
timezone 网站使用的[时区][9]，默认是本服务器使用的时区

#URL

设置项 描述
url 网站中hexo页面所在的网址，用于网页里面的链接
root 网站中hexo页面所在的地址，同样用于网页里的链接
permalink 生成页面时文章网址的[格式][10]
permalink_defaults 文章网址格式中内容的默认值
